# MyVolt
The enterprise dashboard built for professional stations. Designed to work with AzuraCast.

## Features
- OIDC Sign-in
- Hooks in with Authentik (via webhooks - for suspensions/account changes)
- Temporary streamer accounts

## Usage
Install dependencies using your preferred package manager.
```
pnpm i
```
Initialize the project
```
pnpm run dev
```

And you're all done, setup via the web via the first-time setup wizard.
